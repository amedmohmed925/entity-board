# تقرير متطلبات الواجهة الخلفية (Backend API) لصفحات المشاهد (Viewer)

هذا التقرير يفصل نقاط الاتصال (Endpoints) المطلوبة لتشغيل صفحات المشاهد، والتي تركز بشكل أساسي على عرض البيانات المشتركة (Read-only).

---

## 1. لوحة التحكم للمشاهد (Viewer Dashboard)
**المسار:** `/viewer`

### [GET] `/api/viewer/stats`
جلب ملخص الإحصائيات المسموح للمشاهد برؤيتها.
- **الاستجابة:**
  ```json
  {
    "totalSales": { "value": "45,230$", "change": "+12%" },
    "activeUsers": { "value": "1,240", "change": "+5%" },
    "newOrders": { "value": "189", "change": "-2%" },
    "conversionRate": { "value": "3.2%", "change": "+0.5%" }
  }
  ```

### [GET] `/api/viewer/reports/recent`
قائمة بآخر التقارير التي تمت مشاركتها مع هذا المشاهد.
- **الاستجابة:** قائمة كائنات تحتوي على `id`, `title`, `sharedBy`, `timestamp`.

---

## 2. لوحات البيانات المشتركة (Shared Dashboards)
**المسار:** `/viewer/dashboards`

### [GET] `/api/viewer/dashboards`
جلب قائمة بجميع لوحات البيانات التي يمتلك المشاهد صلاحية عرضها.
- **الاستجابة:** `[ { "id": 1, "title": "ملخص المبيعات", "category": "المالية", "lastViewed": "..." }, ... ]`

### [GET] `/api/viewer/dashboards/:id`
جلب البيانات التفصيلية للوحة بيانات معينة لعرضها في مودال المعاينة.
- **الاستجابة:** تشمل الإحصائيات، بيانات الرسم البياني (Bar Chart/Donut Chart)، وقائمة بآخر العمليات (Transactions).

---

## 3. التقارير والتحليلات (Reports)
**المسار:** `/viewer/reports`

### [GET] `/api/viewer/reports`
جلب قائمة بجميع التقارير المتاحة للتحميل أو العرض.
- **الاستجابة:** قائمة تحتوي على `id`, `title`, `date`, `type` (PDF/Excel).

### [GET] `/api/viewer/reports/:id`
جلب محتوى التقرير التفصيلي لعرضه في المعاينة (Preview).
- **البيانات المطلوبة:** جداول بيانات، نصوص تحليلية، واحصائيات رئيسية.

### [GET] `/api/viewer/reports/:id/download`
رابط تحميل ملف التقرير الأصلي.

---

## 4. المساعدة والدعم (Help & Support)
**المسار:** `/viewer/help`

### [GET] `/api/viewer/faqs`
جلب قائمة الأسئلة والأجوبة الشائعة.

### [POST] `/api/viewer/support/request`
بدء جلسة دردشة أو إرسال طلب دعم فني.

---

> [!NOTE]
> جميع نقاط الاتصال أعلاه يجب أن تكون **للقراءة فقط (Read-only)**، حيث لا يمتلك المشاهد صلاحية التعديل أو الحذف أو الإضافة في أي من هذه البيانات.
